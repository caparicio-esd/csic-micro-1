## Fases de un proyecto

{/* CRISP-DM ligero: comprensión del problema → datos → preparación → modelado → evaluación → despliegue → monitorización */}
{/* Iterativo, no lineal; el 80% del esfuerzo está antes del modelo */}

```mermaid
%%{init: {'theme': 'base'}}%%
flowchart LR
    P["Comprensión\ndel problema"]
    D["Comprensión\nde datos"]
    PR["Preparación\nde datos"]
    M["Modelado"]
    E["Evaluación"]
    DE["Despliegue"]
    MO["Monitorización"]

    P --> D --> PR --> M --> E --> DE --> MO
    MO -.->|"iterar"| P
    E -.->|"revisar"| PR
```
